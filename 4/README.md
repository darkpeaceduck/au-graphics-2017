#### Build

```sh
$ ./cmd.sh build
```
(или cmake + make)
#### Run

```sh
$ ./cmd.sh run
```
#### Clean

```sh
$ ./cmd.sh clean
```

#### Deps
GLFW, Assimp, Opengl lib
Установка зависимостей, если есть apt : 
```sh
$ ./cmd.sh deps
```

#### Controls

WASD + Mouse

M - change mode (0 - compose, 1 - hbao only)
